Product Features

 

Some of license manager key features

Variable Products Support

Fully Compatible with WooCommerce variable products, send different license keys/code per variation

Backorders Support

If the license keys stock is low, they can be assigned later to the order allowing the product backorders

License Keys Encryption

All the license/codes in the database are encrypted to make sure they are always safe and secure

Product Bundles Support

Compatible with product bundles. The plugin will deliver license keys for your bundled products.

Software Licensing API

Developers API to license and control software activation and use. License veriﬁcation, and deactivation.

Outstanding Support

We provide fast, professional support. Feel free to ask your questions, and we will answer as soon as possible.

Image License Keys

Delivery license keys in an image format. Steam cards, phone recharge cards, gift cards, etc

Bulk Import/Export

License/codes can be bulk imported to save time. Also, they can be exported all or by status or product

Custom Order Status Support

All native and custom order statuses are supported to give you full control over when license keys are delivered

License Keys Generator

The plugin will generate license keys for your digital products and send them to your customers.

Translation Ready

Translation ﬁles are included in the package so you can easily translate the plugin to your language

Feature Requests
